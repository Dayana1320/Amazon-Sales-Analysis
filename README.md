# Amazon Sales Analysis Using Excel

## Project Overview

This project analyzes Amazon sales data to identify sales trends, customer behavior, product performance, and payment preferences using Microsoft Excel.

## Objectives

* Analyze overall sales performance.
* Identify top-selling products.
* Understand customer location trends.
* Study payment method distribution.
* Evaluate order status patterns.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning Techniques

## Data Cleaning

The dataset was cleaned by:

* Removing duplicates
* Handling missing values
* Standardizing formats
* Verifying data consistency

## Analysis Performed

* Total Sales by Category
* Total Sales by Product
* Sales by Customer Location
* Order Status Analysis
* Payment Method Distribution

## Dashboard Features

* Interactive charts
* Slicers for filtering data
* Easy-to-understand visualizations
* Dynamic analysis capability

## Key Insights

* Certain categories contribute significantly to total sales.
* Customer locations influence purchasing patterns.
* Payment methods vary across customers.
* Order status tracking helps evaluate business performance.

## Conclusion

This project demonstrates the use of Excel for data analysis, visualization, and dashboard creation to support data-driven business decisions.
